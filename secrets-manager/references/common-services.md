# Common Services Reference

## Authentication & Identity

### Descope
- **Variables**: `DESCOPE_PROJECT_ID`, `NEXT_PUBLIC_DESCOPE_PROJECT_ID`, `DESCOPE_MANAGEMENT_KEY`
- **URL**: https://app.descope.com
- **Console**: Settings → Project → Management Keys

### Auth0
- **Variables**: `AUTH0_DOMAIN`, `AUTH0_CLIENT_ID`, `AUTH0_CLIENT_SECRET`
- **URL**: https://manage.auth0.com

### Clerk
- **Variables**: `CLERK_SECRET_KEY`, `NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY`
- **URL**: https://dashboard.clerk.com
